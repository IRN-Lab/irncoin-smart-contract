Overview – IRN Coin (IRN)

IRN Coin is a decentralized digital asset built on the Polygon Proof‑of‑Stake (PoS) network.

This repository contains the official smart contract source code, ABI, and documentation for developers and integrators.

IRN Coin is designed with a fixed total supply, transparent tokenomics, and a roadmap focused on ecosystem growth, utility development, and long‑term sustainability.

Key Information

• Name: IRN Coin

• Symbol: IRN

• Network: Polygon PoS

• Total Supply: 255,000,000 IRN

• Contract Type: ERC‑20 Standard

• Compiler Version: Solidity ^0.8.20

Repository Structure

• /src – Smart contract source code (IRNCoin.sol)

• /abi – Compiled ABI for integrators

• /docs – Documentation (Markdown)
